# Introduction to Statistics Concept Pages

Drop reviewed `.md` concept pages into:

`data/wiki/concept_pages/`

These pages provide compact, student-facing concept summaries for the Shiny proof of concept. They should be reviewed before use in any real course or public deployment.

Recommended authority rules:

1. A licensed, open-license, or instructor-created introductory-statistics text should guide formulas, terminology, and core explanations.
2. Instructor notes or formula sheets can be used as overlays for notation, emphasis, and exam-facing examples.
3. Supplemental materials should add examples or alternate explanations, not silently override the main source.

The pages can support:

- multiple-choice practice
- fill-in-the-blank practice
- choose-the-best-answer practice
- visual explanations and retrieval-grounded tutor responses
